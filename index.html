Subscription troubleshooting

  1. Ensure you have the latest app version installed.
  2. To restore purchases: open the app → Settings → Restore Purchases.
  3. If Subscribe does nothing in TestFlight: sign out of the App Store (Settings → Apple ID → Sign Out), then attempt the purchase and sign in when prompted with your Sandbox Tester credentials.

Testing / Review notes

Sandbox / TestFlight

For testers: create a Sandbox Tester in App Store Connect and use those credentials when prompted during purchase.

Reviewer dev bypass (ephemeral): enter the code qwerty12345 on the paywall to unlock the app until the next app restart.

What to include in a support request

Please include:

  • Device model (e.g. iPhone 14) and iOS version
  • App build number (from Settings → About)
  • A short description of the issue and any console logs if available
